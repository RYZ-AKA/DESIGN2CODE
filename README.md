# Design2Code

## 📌 Description
Design2Code is a web-based drag-and-drop builder that allows users to design layouts visually and export the generated HTML, CSS, and JavaScript code.

This project is currently under active development, and new features are continuously being improved and added.

> ⚠ Currently optimized for desktop use only.

---

##  Current Features
- Drag-and-drop elements
- Visual layout preview (non-live rendering)
- Code generation
- Export & import project / copy generated code
- Basic UI structure

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

##  Planned Features
- Improved drag-and-drop experience
- Better mobile responsiveness
- Enhanced UI/UX
- Component customization options
